# Website Structure Overview

This website consists of three main files that work together to create a responsive, interactive landing page:

## index.html
The main HTML structure file containing:
- Complete page layout with navigation, hero section, features, testimonials, and footer
- Responsive design using Tailwind CSS classes
- Interactive elements like dropdowns, carousels, and tabs
- Integration points for the JavaScript functionality
- Link to external resources and internal CSS/JS files

## styles.css
The stylesheet containing:
- All CSS rules for visual presentation and layout
- Animations and transitions for interactive elements
- Responsive design rules for different screen sizes
- Custom styling for components like chat bubbles, buttons, and cards
- Gradient effects and special visual treatments
- Component-specific styles for the ultra-smooth dropdowns

## script.js
The JavaScript file containing:
- DOM manipulation and event handling
- Typewriter effect for the hero section
- Mobile menu toggle functionality
- Scroll-based animations and parallax effects
- Tab switching functionality
- Ultra-smooth dropdown functionality (exclusive open behavior)
- Carousel functionality for testimonials
- Chatbot widget implementation
- Intersection observers for performance optimization

Together, these files create a modern, interactive website with smooth animations, responsive design, and rich user interactions.
